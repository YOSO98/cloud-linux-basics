##  Analyse Cloud

- Le système est basé sur Debian, une distribution largement utilisée
  sur les serveurs Cloud.
- L’environnement est exécuté sous WSL2, utilisé ici comme
  environnement de développement local.
- Les ressources CPU, mémoire et disque sont suffisantes pour
  héberger des services Cloud légers (web, API, conteneurs).
- Cette configuration simule un serveur Cloud pour des
  travaux d’administration et d’automatisation.
